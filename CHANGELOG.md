# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2025-01-08

### ✨ New Features
- **联系我们页面重设计**: 全新精美的联系我们界面，包含现代化设计和丰富的交互功能
- **用户发布页面优化**: 新增三个标签页（二手商品、餐厅评价、拼车信息），支持分类管理用户发布内容
- **拼车功能增强**: 
  - 重新设计拼车列表和发布界面，提升用户体验
  - 新增客户端筛选功能（出发地、目的地、日期筛选）
  - 优化表单布局和验证逻辑
- **二手商品功能完善**: 
  - 支持编辑和删除用户发布的二手商品
  - 优化审核状态显示和处理逻辑
  - 新增图片上传和管理功能

### 🔧 API Integration
- **新增用户数据接口**: 集成获取用户餐厅评价和拼车信息的后端API
- **二手商品管理接口**: 新增更新和删除用户二手商品的API接口
- **数据状态管理**: 完善各模块的数据获取和状态管理逻辑

### 🎨 UI/UX Improvements
- **联系我们页面**: 
  - 渐变背景和浮动装饰动画
  - 精美的联系方式卡片设计
  - 完整的在线表单功能（暂时移除）
  - 响应式设计适配多种屏幕尺寸
- **拼车界面优化**: 
  - 重新设计表单布局，采用原生Taro组件
  - 优化输入框状态管理和事件处理
  - 添加快捷路线选择和字符计数功能
- **用户发布界面**: 
  - 新增标签页导航
  - 统一的卡片设计风格
  - 改进的空状态展示

### 🐛 Bug Fixes
- **表单输入问题**: 修复拼车发布页面输入框值异常显示的问题
- **状态管理**: 解决多个页面的状态同步和更新问题
- **类型检查**: 修复TypeScript类型定义重复和不匹配的问题
- **样式兼容**: 解决NutUI组件与Taro组件混用时的样式冲突

### 🛠 Technical Improvements
- **代码重构**: 优化多个页面的组件结构和状态管理
- **性能优化**: 改进数据获取和渲染性能
- **类型安全**: 完善TypeScript类型定义和接口规范
- **代码质量**: 修复ESLint警告和代码规范问题

### 📱 Platform Support
- **微信小程序**: 全面支持微信小程序平台
- **响应式设计**: 适配不同屏幕尺寸的设备
- **跨平台兼容**: 保持与Taro多端开发框架的兼容性

- 基础功能模块实现
- UI组件库集成（NutUI React Taro）
